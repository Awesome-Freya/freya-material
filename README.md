# `freya-material`

Freya Material is a library for `freya` that provides Material You style components.

It is currently in early stage of development.

## Progress

- [ ] MD Tokens
  - [x] Colors
  - [ ] Typography
  - [x] Shapes
  - [x] Motion
- [ ] Components
  - [ ] State Layers
  - [ ] Icons
  - [ ] Menus
  - [ ] Text Fields
  - [ ] Tabs
  - [ ] Lists
  - [ ] Dialogs
  - [ ] Time Pickers
  - [ ] Sliders
  - [ ] Top App Bar
  - [ ] Badges
  - [ ] Date Pickers
  - [ ] Carousel
  - [ ] Navigation Drawer

  - [ ] Search
